# DeepLearning Studies

A collection of deep learning notebooks developed during university studies, covering computer vision, time series, recurrent neural networks, and natural language processing using Python and PyTorch.

---

## Notebooks

### Computer Vision

| Notebook | Description |
|----------|-------------|
| `transfer_learning.ipynb` | Image classification using transfer learning with pre-trained models |
| `vit_fine_tuning.ipynb` | Fine-tuning a Vision Transformer (ViT) for image classification |
| `image_segmentation.ipynb` | Image segmentation using deep learning |
| `image_segmentation_adjustments.ipynb` | Improvements and adjustments to the segmentation model |

### Time Series & RNN

| Notebook | Description |
|----------|-------------|
| `time_series_training.ipynb` | Time series forecasting with deep learning |
| `rnn_training.ipynb` | Recurrent Neural Network (RNN) training for sequential data |

### Natural Language Processing

| Notebook | Description |
|----------|-------------|
| `classic_nlp.ipynb` | Classic NLP techniques — tokenization, TF-IDF, text classification |
| `text_similarity.ipynb` | Text similarity using embeddings |
| `bert_fine_tuning.ipynb` | Fine-tuning BERT for text classification tasks |

---

## Tech Stack

- **Python 3**
- **PyTorch** — deep learning framework
- **Transformers (HuggingFace)** — ViT and BERT models
- **Scikit-learn** — classic ML utilities
- **Pandas / NumPy** — data manipulation
- **Matplotlib / Seaborn** — visualization
- **Kaggle API** — dataset retrieval
